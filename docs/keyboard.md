# Keyboard Notes

## Hardware

Apple Magic Keyboard connected over Bluetooth.

## Fedora detection

- Command key = Super / Meta
- Option key = Alt
- Control key = Control
- Right Option = AltGr / ISO_Level3_Shift

## Current decision

Do not globally remap the keyboard yet.

Reason:

- Fedora/KDE recognises the keyboard correctly.
- The real issue is macOS muscle memory while still using a Mac for work.
- Keyboard remapping will be handled later as a focused profile.
