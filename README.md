# AsifOS

A Fedora KDE developer workstation for a Laravel/PHP developer moving from macOS to Linux.

## Goals

- Document my workstation setup
- Make the setup reproducible
- Preserve macOS developer muscle memory where useful
- Build a Laravel, Docker, AWS and AI-ready Linux workstation
- Eventually provide a setup others can reuse

## Current Stack

- Fedora KDE
- Apple Magic Keyboard
- Git + GitHub SSH
- Docker Engine
- Docker Compose
- PhpStorm
- Google Chrome
- Slack
- RustDesk
